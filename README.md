# How to Draw with Affinity
여기에 제가 Affinity에 대해 배운 것을 적겠습니다

## 기본 구조

위쪽 탭을 선택해서, Vector, Pixel, Layout, CanvasAI중 하나를 선택 가능합니다.  
이 탭은 수정이 가능합니다.

## Vector  
+ <B>'shape tool'</B>로 도형을 생성합니다. 상단 바로 여러 옵션을 바꿀 수 있습니다.
+ <B>'move tool'</B>로 도형을 움직이고 늘이는 등의 동작을 합니다.  
+ <B>'node tool'</B>로 도형의 node를 지우거나 추가하거나 변형할 수 있습니다.  
+ <B>'artboard tool'</B>로 또다른 페이지를 만들 수 있습니다. 큰 프로젝트에서 사용합니다.
+ <B>'point transform tool'</B>로 한 점을 기준으로 이동시킬 수 있습니다.
+ <B>'corner tool'</B>로 가장자리 형태를 바꿉니다.
+ <B>'contour tool'</B>로 현재 도형의 크기를 바꿉니다(?).
+ <B>'stroke width tool'</B>로 도형 모서리 두께를 바꿉니다.  
(vector에는 두가지 색이 있습니다. '테두리 색'과 '내부 색'입니다.)
+ <B>'fill tool'</B>로 gradient 색을 넣습니다. 투명도도 조절 가능합니다.
+ <B>'text tool'</B>로 텍스트를 넣습니다. 가장자리도 마찬가지로 조절 가능합니다.
+ <B>'place tool'</B>로 사진을 넣습니다만, 그냥 드래그해서 넣어도 무방합니다.
+ <B>'pen tool'</B>로 선을 그리거나, 도형을 그릴 수 있습니다.  
(펜 툴로 선을 그렸는데 fill color가 채워질 경우, 색을 없앨 수도 있습니다.)
+ <B>'pencil tool'</B>로 더 자유롭게 선을 그을 수 있습니다.   
(fill color를 추가할 수 있으며, 화살표로 테두리 색과 내부 색을 바꿀 수도 있습니다.)   
(스포이드를 드래그해서 색을 가져올 수 있습니다.)
+ <B>'shape builder tool'</B>로 서로 다른 shape들을 드래그해 선택한 뒤, +를 누르고 다시 드래그해 합치거나 -를 누르고 다시 드래그해 뺄 수 있습니다. intersect 모드로 드래그해 선택하면, 새 shape를 만듭니다. 매우 유용하다고 합니다.
+ <B>'backet tool'</B>을 통해, 별개의 선으로 이루어진 도형에 색을 부을 수 있습니다.
+ <B>'QR tool'</B>로 url을 만든 뒤 스캔할 qr코드를 제작할 수 있습니다.
+ <B>'measure tool'</B>로 길이, 영억을 측정할 수 있습니다.
+ <B>'vector crop tool'</B>로 벡터 도형을 자를 수 있습니다. 

## pixel  
이미지를 끌어와 편집하는 것이 주입니다.  


참고로, pixel mode에서는 artboard 크기를 바꿀 수 없습니다. 위쪽 탭을 통해 vector mode로 이동한 뒤에  
'artboard tool'이나 'move tool'을 통해 바꾼뒤 다시 pixel mode로 이동합시다.

+ <B>'object selection tool'</B>로 이미지의 일부를 선택합니다. 브러쉬 모드로도 사용할 수 있습니다.
(위쪽 탭을 통해, 더 섬세하게 자르고 배경을 제거하거나 마스크를 사용할 수 있습니다.)
(D를 눌러 선택을 취소할 수 있습니다.)
+ <B>'gradient fill tool'</B>로 gradient color를 지정할 수 있습니다.
+ <B>'brush tool'</B>로 선을 그립니다. 상단 탭과 우측 탭으로 설정을 다양하게 바꿀 수 있습니다.
+ <B>'adjustment brush tool'</B>로 다양한 효과를 가진 브러시를 사용할 수 있습니다.
+ <B>'filter crop tool'</B>로 일정 영역에 효과를 주는 브러시를 사용할 수 있습니다.
(효과를 주는 브러시들은 전부 원본을 바꾸지 않습니다. 원할 때 효과 레이어를 지워 효과를 없앨 수 있습니다.)
+ <B>'inpainting brush tool'</B> 그림의 미세한 부분을 없앨 수 있습니다.

## layout
파워포인트처럼 페이지를 추가할 수 있습니다. 출판물 용도입니다.
+ <B>'frame text tool'</B>로 텍스트를 넣고, 페이지간 자연스러운 연결을 할 수 있습니다.
+ <B>'table tool'</B>로 표를 넣을 수 있습니다.


windows > general > assets에서 에셋 창을 볼 수 있습니다.
file > export에서 다양한 포맷으로 파일을 내보낼 수 있습니다.
